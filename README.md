## 本仓库的意图

- 便于 **本地或自托管** 打开使用，不依赖特定站点是否在线。
- 保留可读的静态资源结构（HTML / CSS / JS）。

## 使用方法

1. 确保 `index.html`、`main.css`、`main.js` 在同一目录（本仓库已如此）。
2. 用浏览器打开 `index.html`。
3. 在首页点击 **「开始测试」**，逐题选择后点击 **「提交并查看结果」**。
4. 可通过 **「返回首页」** / **「重新测试」** / **「回到首页」** 切换流程。
